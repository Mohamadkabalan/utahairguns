1. SETUP INSTRUCTIONS AND USAGE GUIDE
- Step 1: Unzip file leurlrewrite.zip.
- Using your FTP software, upload the folder 'leurlrewrite' to your /wp-content/plugins
- Log in to your WordPress Admin panel.
- Go to : Plugins
- Find plugin 'Cart Migration Url Plugin'
- Click 'Activate'
